Overview
Animations can enhance your app by adding movement and interactivity. By using the animations from our Animo library, you can download and add the effect to your own code on your local machine
Installation and usage
Installation

To install your animation from Animo is easy and requires little to no code experience. Our team understands that a lot of users like platforms where they can code without interruptions and distractions and this is exactly what Animo provides. Easy to use and understand.

To install, simply :

  1. Download the library
  2. Add the animo.css file to your folder (the same folder as your HTML file)
  3. copy and paste the code below into the head element of your HTML code

link href=”animo.css” rel=”stylesheet” type=”text/css”
Usage
For each element to be animates, udate or add the class name required forthe specific animation of your choice and then run your html code to view the animation.
