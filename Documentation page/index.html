This is a detailed and comprehensive guide of how to use our animation library.

Animo

Animo Library is a third party open-source library that contains pre-made animation files(codes). They are cross-browser animations made for you to use in your projects.

Bounce

Fade In

This adds the fade in effect to your web element

Flip

This adds the effect to show the front and back of an element.

Swing

This adds the effect that causes an element to move back and forth or from side to side while suspended or on an axis.

Tada

This adds the effect to move an element from side to side
